## [1.0.0] - 2026-02-14

### Added
- Initial release of OpenCAP AI Chat Studio.
- Multi-model configuration (Add, Edit, Delete models).
- Conversation management (New Chat, Delete, Edit Title).
- AI Streaming simulation (character-by-character typing).
- Dark/Light/System theme support.
- Message actions (Copy, Retry).
- Support for SAP AI Core Deployment URLs and System Prompts.
- Integrated Busy Indicators and Auto-scrolling.
- Full English localization.
- Apache 2.0 License and Code of Conduct.
- Latest versions of SAP CAP (@sap/cds 9.7.1) and UI5 dependencies.
